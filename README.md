# TerraStrike Quiz — Dashboard

Dashboard aggregata (sola lettura) delle risposte del quiz TerraStrike.
Mostra solo dati aggregati: nessuna email o dato personale.

- Sorgente dati: viste `dash_*` su Supabase (publishable key, sola lettura).
- Pagina unica `index.html`, nessuna build, nessun segreto.
